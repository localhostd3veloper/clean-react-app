# Clean React App

**this is a cleaner template than npm's create-react-app**

1. Just copy the contents to your project repository
2. Open Terminal
   > `cd path/to/repo`
3. Okay, You have successfully navigated to the project folder
   > `npm install`
4. One Last Thing!
   > `npm start`
